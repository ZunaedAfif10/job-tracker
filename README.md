## 📌 About The Project

Job Tracker is a **frontend-only web application** designed to help users manage and organize their job applications in a simple and structured way.

It is built using **HTML, Tailwind CSS, and Vanilla JavaScript**, focusing on core frontend fundamentals like DOM manipulation, event handling, and dynamic UI rendering.

The goal of this project is to simulate a real-world job tracking system while keeping the implementation lightweight and beginner-friendly.

---

## 🎯 Why This Project?

Tracking job applications manually can become messy and disorganized.

This project was built to solve that problem in a simple way by allowing users to:

* Add job applications
* View and manage entries
* Keep track of applied jobs in one place

It also serves as a strong practice project for mastering core JavaScript concepts without frameworks.

## 🚀 Getting Started

Follow these steps to run the project locally on your machine.

---

### 📥 Clone the Repository

```bash id="c2kq9p"
git clone https://github.com/ZunaedAfif10/job-tracker.git
```

---

### 📂 Navigate to Project Folder

```bash id="k9m3qv"
cd job-tracker
```

---

### ▶️ Run the Project

Since this is a **frontend-only project**, no installation or build process is required.

#### Option 1: Open Directly

* Open `index.html` in your browser

#### Option 2: Use Live Server (Recommended)

If you're using VS Code:

1. Install the **Live Server** extension
2. Right-click `index.html`
3. Click **"Open with Live Server"**
